# Игра «Гений-идиот» 
Консольная и WinForms-версия шуточной викторины, где игрок отвечает на вопросы, а программа оценивает результат по шкале от «Идиот» до «Гений».

# Технологии
C# – язык программирования.

.NET 9.0

Windows Forms – для графического интерфейса.

System.Text.Json – работа с JSON (сериализация/десериализация).

Консольное приложение – для демонстрации базовой логики.

# Запуск
Клонировать репозиторий:

bash
git clone https://github.com/twixkox/GeniyIdiot.git
